# 🌱 SproutCircle

A warm, simple, and tablet-friendly daycare management platform designed to help caregivers stay organized throughout the day while keeping parents informed.

SproutCircle makes it easy for daycare staff to manage enrolled children, track attendance, record daily activities, and share updates with parents in real time through a clean and friendly interface.

---

## ✨ Features

### 👶 Child Management
- View all enrolled children from a central dashboard
- Add new child profiles
- Edit existing child information
- Store important details such as:
  - Name
  - Age
  - Parent/Guardian Information
  - Allergies
  - Emergency Contact Details
  - Profile Photo

### ✅ Attendance Tracking
- Check children in at arrival
- Check children out at pickup
- Quickly view current attendance status
- Color-coded status badges for easy visibility

### 📝 Daily Updates
Record important events throughout the day, including:

- 🍽 Meals and snacks
- 😴 Nap schedules
- 🚽 Bathroom updates
- 😊 Mood tracking
- 🎨 Activities completed
- ⚠ Incident or injury notes
- 💬 General observations

Updates are designed to be entered quickly from a tablet without interrupting caregiver workflows.

### 👨‍👩‍👧 Parent Portal
Parents can securely access:

- Their child's profile information
- Daily activity logs
- Attendance status
- Meal updates
- Nap reports
- Mood summaries
- Incident notifications

Parents can only view information related to their own child.

---

## 🎯 Project Goals

SproutCircle focuses on simplicity over complexity.

Many childcare management tools feel like enterprise software packed with unnecessary features. SproutCircle is designed to feel approachable, calm, and easy to use during a busy daycare schedule.

Key priorities:

- Fast interactions
- Large touch-friendly controls
- Minimal learning curve
- Clear visual hierarchy
- Friendly and welcoming design

## 🏗 Dashboard Overview

### Staff Dashboard

The main dashboard provides:

- Child attendance overview
- Today's updates
- Quick actions
- Search and filtering
- Child profile management

### Child Profile Page

Each profile contains:

- Personal information
- Parent information
- Medical notes
- Attendance history
- Daily update timeline

### Parent Dashboard

Parents see:

- Child profile summary
- Daily reports
- Attendance information
- Recent updates

No staff administration tools are visible in the parent view.
